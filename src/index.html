WEB API Playground

Background Task

These ghosts are moved in a background task (requestIdleCallback) and drawn in requestAnimationFrame. Since the movement isn't critical to this page we can only animate if we have the processing power to do so.

Broadcast Channels

Shy ghost is shy. Open another window to this page and send shy ghost a message. Maybe that will make shy ghost less shy.

CSS Counters

Click the button to add an item. The numbering is all added through CSS.
